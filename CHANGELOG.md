# Changelog

## 1.0.0 (2026-07-11)
- Initial release
- Polling-based file watcher (no dependencies)
- Recursive and flat directory watching
- Glob pattern filtering
- Debounce mechanism to prevent double-triggers
- Command templates with {file}, {event}, {dir} variables
- Dry-run mode for safe testing
- JSON output for CI integration
- Cross-platform: Linux, macOS, Windows, Docker
